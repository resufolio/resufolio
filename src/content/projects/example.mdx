# Example Project

It's an example project to demonstrate the structure of a project page.

## How to change the content?

You can change the content of this page by editing the file `src/content/projects/example.mdx`.

## How to add a new project?

You can add a new project by creating a new file in the `src/content/projects` directory. The file should have the `.mdx` extension.

## Structure example

The structure of a project page is defined by the front matter and the markdown content. The front matter is a YAML block that contains metadata about the project. The markdown content is the body of the page.

```markdown
---
title: Example Project
tags:
- TypeScript
- Library
- NPM
description: It's an example project to demonstrate the structure of a project page.
links:
  - source: GitHub
    href: https://github.com/teles/array-mixer
  - source: NPM
    href: https://www.npmjs.com/package/array-mixer
image:
  src: /images/logos/array-mixer-article.svg
  alt: Array Mixer
  width: 800
  height: 400
skills:
  - TypeScript
  - Jest
  - Node.js
  - NPM
---
```
